# DDDBrisbane-Building-Trustworthy-AI

Code is Life: Building a Trustworthy Generative AI Solution at DDDBrisbane 2025

## 🌟 Overview

This is a comprehensive learning resource on building trustworthy and ethical AI systems. The site is built with MkDocs and Material theme, featuring a modern, responsive, and accessible design.

## 🚀 Quick Start

### Local Development

1. Install Python dependencies:
```bash
pip install -r requirements.txt
```

2. Run the development server:
```bash
mkdocs serve
```

3. Open your browser to `http://127.0.0.1:8000`

### Building the Site

Build the static site:
```bash
mkdocs build
```

The built site will be in the `site/` directory.

## 📚 Content Structure

The course includes 50 chapters organized into 10 sections:

1. **Getting Started** - Foundations of trustworthy AI
2. **Core Concepts** - Key principles and frameworks
3. **Technical Foundations** - ML/DL fundamentals
4. **Generative AI** - Modern GenAI technologies
5. **Safety and Security** - Protecting AI systems
6. **Testing and Validation** - Quality assurance
7. **Responsible AI** - Governance and compliance
8. **Implementation** - Building production systems
9. **Real-World Applications** - Industry use cases
10. **Advanced Topics** - Cutting-edge techniques

## 🎨 Design Features

- **Color Scheme**: Purple → Blue → Green gradient
